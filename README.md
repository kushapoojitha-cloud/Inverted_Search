🧠 Inverted Search 
📌 Project Description
This project implements an Inverted Search Engine using the C programming language. 
It creates a database that maps words to the files in which they appear, allowing efficient searching of words across multiple files.

🚀 Features
Create an inverted index from multiple text files
Store words using hashing technique
Display database in structured format
Search for a word across files
Update database with new files
Efficient retrieval using linked lists

🛠️ Technologies Used
C Programming
Data Structures (Linked Lists, Hash Tables)
File Handling

⚙️ How It Works
Read input files
Break content into words
Store words in hash table
Each word points to file list
Each file stores word count

📌 Applications
Search engines
Document indexing
Text analysis systems
